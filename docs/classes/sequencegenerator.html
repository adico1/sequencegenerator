Class SequenceGenerator

Migrated to JS by adico original article: https://www.callicoder.com/distributed-unique-id-sequence-number-generator/

Distributed Sequence Generator. Inspired by Twitter snowflake: https://github.com/twitter/snowflake/tree/snowflake-2010

This class should be used as a Singleton. Make sure that you create and reuse a Single instance of SequenceGenerator per node in your distributed system cluster.
